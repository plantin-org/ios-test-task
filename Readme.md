# Тестове завдання

## Опис

Уявімо, що при розробці додатку стикнулися з необхідністю працювати з зображеннями. Для цього визначили універсальний протокол, реалізацію якого можна буде безпечно використовувати в будь-якому місці додатку. Суть протоколу проста - він має за певним URL завантажувати зображення і повертати їх у вигляді `UIImage`. При цьому хочеться, щоб реалізація була максимально легка та не потребувала сторонніх бібліотек.

## Завдання

Реалізувати протокол `ImageFetcher`:

```swift
protocol ImageFetcher {
    func fetchImage(from url: URL) async -> UIImage?
}
```

## Вимоги

Реалізація має задовольняти наступним вимогам:

- Запит на отримання зображення за певним URL має бути виконаний максимум один раз, повторні виклики функції не мають призводити до нових завантажень.
- Кешування результатів можна робити просто в пам'яті.
- Використання має бути потокобезпечним та не блокувати головний потік.

## Перевірка

Для перевірки правильності виконання треба запустити функцію `validate(using fetcher: ImageFetcher)`, передавши туди інстанс вашої імплементації протоколу.

**⚠️ Код функції `validate` змінювати не можна!**

## Обмеження

Можна використовувати будь-які інструменти та AI, але не можна користуватися сторонніми бібліотеками.
